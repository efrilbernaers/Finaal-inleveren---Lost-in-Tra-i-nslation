lesweek 5: 16/10/2025 - 23/10/2025

ChatGpt

Tijdens het werken met ChatGPT heb ik de resultaten meegenomen naar de les om verder te experimenteren en te leren hoe mijn ontwerpen vertaald kunnen worden naar HTML. Dit was een eerste stap om mijn digitale vaardigheden uit te breiden.

Leren werken met html

Ik heb Visual Studio Code geïnstalleerd, een programma waarmee ik code kan schrijven, testen en aanpassen in een overzichtelijke omgeving.

In de les hebben we een zip map gedownload en hiermee de HTML structuur aangemaakt.

Daarnaast ben ik bezig geweest met het aanpassen van fonts. Hiervoor heb ik Google Fonts gebruikt, wat me toeliet om de lettertypes aan te passen.

Ook heb ik gewerkt aan documentkleuren, waarbij ik kleuren heb gekozen die goed bij mijn ontwerp passen en de leesbaarheid verbeteren. Dit hielp om de schermen niet alleen functioneel, maar ook visueel aantrekkelijk te maken.

Logboek

In mijn logboek heb ik stap voor stap bijgehouden hoe ik mijn HTML aanvulde met de informatie uit mijn ontwerpdocumentatie van alle voorgaande weken.

Daarnaast heb ik GitHub en Netlify geïnstalleerd. GitHub gebruik ik om mijn code te bewaren en versiebeheer bij te houden, terwijl Netlify ervoor zorgt dat mijn schermen online gepubliceerd kunnen worden, zodat ik ze overal kan bekijken en testen.
