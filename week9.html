lesweek 9: 20/11/2025

Tijdens de lessen over Tailwind heb ik de kennis die ik eerder opgedaan had verder uitgewerkt en toegepast op mijn project.

Hiervoor heb ik de repository BP_HTML_09TWND-main van GitHub gedownload en geopend in Visual Studio Code.

Dit gaf me een werkbare omgeving om de HTML-structuur en de styling naar Tailwind om te zetten.

Om alles lokaal te kunnen draaien, moest ik Node.js installeren.


Vervolgens heb ik in de terminal van Visual Studio Code het commando npm i uitgevoerd om alle benodigde packages te installeren.

Omzetten HTML naar Tailwind

De ontwerpdocumentatie van de HTML diende als leidraad om de volledige structuur en body van de bestaande pagina’s om te zetten naar een Tailwind-versie.

Dit betekende dat alle kaders, layout-elementen en styling opnieuw moesten worden opgebouwd met Tailwind-classes, zoals marges, padding, kleuren en fonts.


Drie kernschermen (vertrekbord, perronscherm, wagonindelingsscherm) te integreren in een interactief prototype.

Het project omvat drie kernschermen die essentieel zijn voor de gebruikerservaring (ziet u rechtsboven in de foto):.

→ Vertrekbordscherm(hoofdscherm) – overzicht van vertrektijden, vertragingen en perrons.

→ Perronscherm – details over de perron waar je een trein kunt opstappen.

→ Wagonindelingsscherm – informatie over de indeling van de treinwagons (zoals;overzicht van rijtuigen, faciliteiten en treinindeling).


Daarnaast zijn er nog meerdere schermen toegevoegd voor interactie en gebruiksgemak:

Linksboven:een scherm waarin gebruikers een eigen traject kunnen plannen (bijvoorbeeld van Sint-Niklaas naar Antwerpen-Centraal).

Onderaan: vier schermen voor algemene navigatie en acties:

→ Hoofdmenu

→ Favoriete trajecten

→ Ticket kopen

→ Aangekochte tickets

Om het prototype volledig interactief te maken, heb ik alle interacties tussen de schermen toegevoegd, zichtbaar met lijntjes die de navigatiestromen aangeven. Dit zorgt ervoor dat gebruikers kunnen zien hoe het scherm van het ene naar het andere verandert wanneer ze bijvoorbeeld een traject plannen of een ticket kopen.
